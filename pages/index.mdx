# Introduction
 
Welcome to Satyam APIs! This is a basic APIs that you can use it as a starting point for your blog projects :)

## What is this API for?

A **simple**, **useful** and **flexible** API that retrieves the blogs from database. You can use this for your project free without need to write blog posts.

## Get Started

To get started you'll need an API key. They're free while you are in development.

Free API is available here: [https://api.satyamregmi.com.np/](https://api.satyamregmi.com.np/).
